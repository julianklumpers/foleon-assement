## Foleon assement

### Requirements

> Node.js version >=12.0.0.

### Install

> `yarn install` or `npm install`

### Run application

> `yarn dev` or `npm run dev`

### Build and run application

> `yarn build` then `yarn serve`
> or
> `npm run build` then `npm run serve`

### Tests

> `yarn test` or `npm run test`
